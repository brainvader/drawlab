# DrawLab

**アーティストのための、アーティストによる、キャラクターデザインツール。**

> AIは絵を生成しない。あくまでアーティストのアシスタントに徹する。

---

## これは何？

DrawLab はアニメ・漫画風キャラクターのデザインを支援するデスクトップアプリです。

よくある「AIがイラストを生成するツール」とは正反対の思想で作られています。**描くのは人間**。AIはパーツの検索補助・フィードバック・形状の微調整アシストのみを担います。

---

## 機能

### 📖 パーツ図鑑

手描きのパーツ（目・眉・鼻・口・髪など）を登録・管理するデータベース。

- カテゴリ・印象・タグによる絞り込み
- **自然言語検索** — 「クールな目」「初心者向けの口」などで検索するとLLMがクエリを解釈してFTS5全文検索に繋ぐ
- パーツごとに「描き順アニメーション」と「ガイド線トレース練習」が付属
- パーツはアーティスト自身が手描きして登録する

### ✏️ シェイプエディター

基本形状をコントロールポイントで調整するベクターエディター。現在は**目**に対応。

- 5つのドラッグ可能なコントロールポイント（目頭・目尻・上ライン頂点・下ライン底点・瞳孔中心）
- 右目はリアルタイムでミラーリング
- パラメーター調整（眉間の幅・目の角度・開閉・虹彩サイズ・瞳孔サイズ・まぶた線幅）
- プリセット（基本・たれ目・つり目・大きい目・ジト目）
- **Undo / Redo**（⌘Z / ⌘⇧Z）
- **AI指示** — 「つり目にして」「目尻を下げて」などの自然言語でコントロールポイントが動く

### 🎨 キャラクリエイター

図鑑に登録されたパーツを組み合わせて2Dキャラクターを作るツール。

- カテゴリ別パーツ選択
- SVGレイヤーによるパーツの重ね合わせ表示
- **三面図プレビュー**（正面・横・後ろ）
- 髪色・肌色のカスタマイズ

## コアコンセプト：AIはアシスタント

| ✅ AIがやること            | ❌ AIがやらないこと    |
| -------------------------- | ---------------------- |
| 検索クエリの意図解釈       | パーツ・絵の生成       |
| シェイプ調整の指示受付     | キャラクターのデザイン |
| 描いた線へのフィードバック | ストロークの代筆       |
| メタデータの補完提案       | 自動着色・自動仕上げ   |

## 技術スタック

| レイヤー       | 技術                              |
| -------------- | --------------------------------- |
| デスクトップ   | Tauri 2.0 (Rust)                  |
| フロントエンド | React 18 + TypeScript + Vite      |
| スタイリング   | Tailwind CSS                      |
| ローカルDB     | SQLite + FTS5（tauri-plugin-sql） |
| 状態管理       | Zustand                           |

データはすべてローカルに保存されます。クラウドへの送信は LLM への検索クエリのみです。

## スコープ外

**StrokeLab**（ストロークレコーダー）は別プロジェクトとして分離しています。描き方データの収集・公開・AIモデルへの学習データ化は DrawLab のスコープ外です。

## 開発方針

このプロジェクトは **PPAC Protocol v2.7** と **TaaC（Test as a Context）** に基づいて開発されます。

- 1チャット = 1工程
- テストファイルがコンテキストのSSOT
- 実装前に必ず公式ドキュメントを調査・グラウンディング

詳細は [ARCHITECTURE.md](./ARCHITECTURE.md) を参照してください。
